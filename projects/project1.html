Iris flower classification

Iris flower classification is a very popular machine learning project. The iris dataset contains three classes of flowers,Versicolor, Setosa, Virginica, and each class contains 4 features, Sepal length, Sepal width, Petal length, Petal width. The aim of the iris flower classification is to predict flowers based on their specific features.

I used various tools to build this project which is used to classify different species of this flower and finally used logestic regression for the classification purpose and achive the accuracy of 98% in differentiaton of various data points to classify its species.

Feel free to check out the Project document file by visiting to my github page.
